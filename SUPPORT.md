# Support

## Getting Help

- Documentation: https://superagenticai.github.io/rlm-code/
- Issues: https://github.com/SuperagenticAI/rlm-code/issues
- Security reports: info@super-agentic.ai

## Before Opening an Issue

1. Confirm you are on a supported Python version (`3.11+`).
2. Run `rlm-code /doctor` or `/sandbox doctor` and include output.
3. Include reproducible steps, model/runtime used, and relevant logs.

## Scope

- `bug`: incorrect behavior, crashes, regressions
- `enhancement`: feature requests and UX improvements
- `documentation`: missing/incorrect docs

Issues without reproduction details may be closed until details are provided.
